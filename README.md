# 디지털 취약계층을 위한 금융 서비스 
시각장애인과 같은 디지털 취약계층을 위해 송금, 계좌조회 등의 금융 서비스를 편하게 음성으로 
사용할수 있게 하는 프로젝트 입니다.

## 사용된 기술 스택

- `JAVA 11`
- `Spring Boot 2.7.17`
- `Thymeleaf` ,`JPA`
- `MySQL 8.0.27`
- `CHAT GPT API` ,`WEBSOCKET`
